Lumine is a collection of modular libraries that provide the foundation for the Lararust framework. Each library is a separate crate that can be used independently.

## Crates

*   `lumine-http`: HTTP utilities for the Lumine libraries.
*   `lumine-support`: Support utilities for the Lumine libraries.
*   `lumine-cli`: CLI utilities for the Lumine libraries.
*   `lumine-database`: Database utilities for the Lumine libraries.
